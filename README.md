This project requires a PHP and MySQL environment (e.g., XAMPP or MAMP).

1. Database Setup
Start Apache and MySQL servers.

Create a new database named budget_tracker then create the necessary tables (e.g., Users).

2. Configuration
Open db.php.

Ensure the database connection credentials match your local setup (e.g., $username = "root", $password = "").

3. Run the Application
Place the project folder into your web server's root directory (e.g., C:\xampp\htdocs\ ).

Open your browser and navigate to: http://localhost/budget-tracker/
